# vue-admin

## 项目启动
```
# 前端项目依赖
npm install
# json-server
npm install -g json-server
```

### 编译开发
```
#根目录文件夹下
npm run serve
#mock文件夹下
json-server db.json
```

### 打包处理
```
npm run build
```
